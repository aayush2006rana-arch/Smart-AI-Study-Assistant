    What is Machine Learning and how does it differ from traditional programming?
    Machine Learning is a subset of AI that allows systems to automatically learn and improve from experience without being explicitly programmed. Traditional programming uses explicit rules, while ML uses statistical patterns learned from data to make decisions.
    MCQMediumShow Answer ↓
    Which algorithm is NOT a supervised learning algorithm?

    A) Linear Regression   B) K-Means Clustering   C) Decision Tree   D) SVM
    ✅ Correct Answer: B) K-Means Clustering — it is an unsupervised learning algorithm used for clustering unlabeled data. The others (Linear Regression, Decision Tree, SVM) are all supervised learning algorithms.
    TheoryHardShow Answer ↓
    Explain the backpropagation algorithm in neural networks with mathematical derivation.
    Backpropagation is an algorithm for computing gradients of the loss function with respect to weights. It applies the chain rule of calculus to propagate error signals backward through the network layers.

    Steps: (1) Forward pass to compute output, (2) Compute loss L = -Σy·log(ŷ), (3) Compute ∂L/∂w for each weight using chain rule: ∂L/∂w = ∂L/∂a · ∂a/∂z · ∂z/∂w, (4) Update weights: w = w - α·∂L/∂w
    InterviewMediumShow Answer ↓
    How would you handle a dataset with 80% missing values in a key feature?
    Approach: (1) Analyze missing value pattern (MCAR, MAR, MNAR), (2) If >50% missing: consider dropping the feature unless it's crucial, (3) If domain-important: use advanced imputation — KNN imputation, model-based imputation, or multiple imputation, (4) Create a binary indicator feature for "was_missing", (5) Validate imputation strategy with cross-validation to prevent data leakage.
    VIVAHardShow Answer ↓
    What is the bias-variance tradeoff and how do you manage it in practice?
    Bias-Variance Tradeoff: High bias = underfitting (model too simple), High variance = overfitting (model too complex). Total Error = Bias² + Variance + Irreducible Noise.

    Management: Use cross-validation to detect overfitting, regularization (L1/L2) to reduce variance, ensemble methods (bagging reduces variance, boosting reduces bias), and learning curves to diagnose bias vs variance issues.
